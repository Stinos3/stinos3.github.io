3D Pong — FPGA Game Console
Fully functional 2-player 3D Pong system with wireless controllers, FPGA logic and custom PCBs. Team project @ Avans.
FPGA/VHDLKiCadRF 433MHzBasys-3
About
Built as a 2nd-year EE project at Avans. Two-player console with wireless bats operating at 10m+ range. My role covered hardware/system design, RF input chain, power & status indicators, FPGA I/O integration, and bat PCB design — including VHDL collaboration for game logic.
Key Hardware
RF link via RX/TX 2A-433-64 with LICAL-EDC-DS001 encoder/decoder. USB-C power path with BQ24090, XC9140A501MR 5V boost, and MCP6541T low-battery detection. Change detection using 74HC574 + 2×74HC266 + 74HC30. Basys-3 FPGA for 1v1 and wall-mode control. LED indication via 74HC2G14 + CD4073 + RGB.
